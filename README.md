# ⚙️ Functional Treat - Data Processing & Analysis Tool

## 📌 Description
Functional Treat is a Python-based menu-driven application that demonstrates the use of functions, recursion, and built-in operations for data analysis.

It allows users to input data, perform statistical analysis, filter values, sort data, and calculate factorial using recursion.

---

## 🚀 Features
- 📥 Input dataset
- 📊 Display summary (min, max, avg, sum)
- 🔁 Factorial using recursion
- 🔍 Filter data using threshold (lambda)
- 🔢 Sort data (ascending/descending)
- 📈 Display statistics

- Python (Core + Functional Concepts)

  - Lambda

  - Lists & Dictionaries
